!!! abstract "Overview"

    Python 是我的主要开发语言，虽然我学习的第一门语言是 Java 🤪。

    该章节记录 Python 学习过程中的知识沉淀，包括但不限于 Python 进阶、常用类库以及相关框架。

    - ***Python Pro***

        记录 Python 语言的高级特性与底层实现，包括对象模型、内存管理、并发编程、运行机制以及标准库设计等内容，可能会比较繁杂。
    
    - ***Data Process***

        利用 Python 进行数据处理和可视化是一个不错的选择，Data Process 是进行数据分析的基础，我希望有一天我可以成为一名数据分析师，
        也许并不是作为主要职业，但数据分析能力会让人的眼睛更加明亮

    - ***FastAPI***

        FastAPI 是最受欢迎的 Python Web 开发框架，它支持异步引擎，我的第一个 Web 程序便是通过它创建的，它给了我学习计算机的第一份成就感。

    - ***Asyncio***

        asyncio 在 python3.4 起加入的新特性，它支持开发者基于协程编写异步代码

    - ***Pydantic***

        Pydantic 是用于数据校验的库，它被 FastAPI 集成
    
    - ***SQLAlchemy***

        SQLAlchemy 允许开发者使用 Python 语句操作数据库，它分为 SQLAlchemy Core 和 SQLAlchemy ORM 两个模块

    - ***httpx***

        也许你听过 requests ，httpx 比它更加强大

    - ***uv***

        - 一款新的 Python 包管理工具
